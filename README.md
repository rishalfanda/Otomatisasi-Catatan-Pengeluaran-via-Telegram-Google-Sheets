# Otomatisasi-Catatan-Pengeluaran-via-Telegram-Google-Sheets
Otomatisasi pencatatan pengeluaran menggunakan n8n, Telegram Bot, Google Gemini Vision, dan Google Sheets. Workflow ini menerima foto struk belanja via Telegram, mengekstrak informasi menggunakan AI, lalu menyimpannya secara otomatis ke Google Sheets. Juga mendukung input manual dan laporan pengeluaran (harian, mingguan, bulanan).
